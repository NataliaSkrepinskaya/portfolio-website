UI Project Fitted Application

The Task

The task is to create an app for people who are new or returning to a sports activity.

Users can search and view routines, guides, daily challenges, and other information. They can also keep a schedule by adding sessions to their personal calendar.

The app can be used whenever users like: at home, on the beach, in the gym, in the park.

It helps to become healthy and enjoy the associated benefits: better mood, weight management, reduce risks of illness, learning something new.

Workouts should be fun and suited to from beginner to advanced users.

Interactions

When I Developed mid-fidelity prototypes I signed interactions such scroll between Home Page and Menu, clicks and transitions between the pages, hidden functions. At this step already I added UI elements as a circle diagram for user's usability.

Moodboard

Why I choosed this moodboard

Orange colour promotes a feeling of general health and emotional energy. The value of the orange colour is stimulating, bright.
Blue - loyalty, communication.Blue is the favorite color for most people. Blue creates an excellent color combination with shades of gray.
When using the blue colour as the base color, the orange color is perfect as an active color.
Typography Archivo Black/Archivo Narrow/Arial are an excellent combination for a sports theme.

Used Tools

For creating high-fidelity prototypes I used Sketch and InVision. I think that is a brilliant idea to create interactive prototypes for clearly understanding the scrolls, clicks, transitions as for usability testing as for showing to stakeholders as a prototype project.

Style Guide

For UI project I must take attention to icons which shown together and it will help with visual consistency. I have the unified range of sizes for it.
One of important part is UI Elements and this project requires its individual elements. Main and secondary navigation, primary and secondary buttons, CTA buttons etc. Forms and fields. Additional elements as calendar, patient timeline, patients lists of done exercises, graphs, diagrams.
